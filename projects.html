Crown Saving

I design Crown Saving as a popup tool for web browsers to help users save money. This tool worked similar to "Honey" which helps users find discount but rather than helping the users spend money, I designed a tool that alerts the user if his balance reaches a point that the user can set to a budget. This will help user be more aware of their spendings.
